## Setup

### Run Project

```bash
python3 manage.py runserver
```

### add new module

```bash
python3 manage.py startapp polls
```

### run migration

```bash
python3 manage.py makemigrations
python3 manage.py migrate
```

### create super admin

```bash
python3 manage.py createsuperuser --username=nadim --email=nadim.sheikh.07@gmail.com
```
